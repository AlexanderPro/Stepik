Вопрос:
Отметьте верные утверждения

Ответ:
Драйвер -- это программный компонент
Драйвер устройства может быть связан с именем в файловой системе
Устройство "мышь" может иметь соответствующее ему имя в файловой системе
Приложение может получить доступ к драйверу устройства через файл
Драйвер может быть связан с виртуальным устройством
