Вопрос:
Дано натуральное число n. Напишите рекурсивную функцию, которая возвращает строку чисел от 1 до n, разделенных пробелом.
Основная сложность задачи заключается в том, что ее необходимо решить с использованием рекурсии. Кроме платформы Stepik проверять ваше решение будут в том числе и другие студенты, с помощью механизма рецензирования.
Чтобы получить максимальный балл за задачу, вам необходимо выполнить следующие условия:
Верно решить задачу (убедитесь, что задача подсвечена зеленым цветом)
Убедиться еще раз, что задача решена с помощью рекурсии (это единственный критерий, по которому вас будут оценивать другие студенты)
Воспользуйтесь кнопкой "Отправить это решение на рецензию". Внимание! Отправить решение на рецензию можно только один раз!
?Проверьте решения минимум трех других студентов. 
Дождитесь, пока ваше решение не проверят.

Ответ:
function my_function(n) {
   return n == 1 ? n.toString() : my_function(n - 1) + ' ' + n;
}
